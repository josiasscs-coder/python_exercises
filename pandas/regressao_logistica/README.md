## Implementar um script de Python que utilize o dataset “titanic_v2.csv”.

Este é um projeto para construir um modelo de regressão logística, com o intuito de prever se um passageiro sobreviveu ou não ao trágico desastre do Titanic de acordo com a classe com que se encontravam e tambem com o género da pessoa. 

## Estrutura do projeto

caso_pratico.ipynb
README.md
Requirements.txt

## Como correr o projeto

1. Abrir um terminal
2. vai ate a pasta do projeto
3. instalar as bibliotecas que se encontram no Requirements.txt
4. executar o código

## Dependências

Instalação do Pandas, Numpy, statsmodels.api as sm
e statsmodels.formula.api as smf
