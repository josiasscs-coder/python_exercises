# Juntar várias páginas de varios PDFs em um só ficheiro utilizando a biblioteca PyPDF2

Este é um projeto em Python que permita a manipulação de PDF, nomeadamente extrair e combinar páginas entre si para formar um novo PDF e adicionar marca de água a paginas selecionadas.

## Estrutura do projeto

- page_merger_codigo.py
  README.md
  Requirements.txt
  
## Como correr o projeto

1. Abrir um terminal
2. vai ate a pasta do projeto:
```bash
cd page_merger
```
3. instalar a biblioteca PyPdf2 (caso ainda não o tenhas):
```bash
pip install PyPDF2
```
4. executar a aplicação: 
```bash
python3 page_merger.py
```

A aplicação será aberta automáticamente no browser
